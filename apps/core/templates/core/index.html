Django'yu Keşfedin

Django, hızlı geliştirmeye ve temiz, faydacı tasarıma olanak sağlayan bir Python web çatısıdır.

İnternet üzerinden binlerce geliştiricinin desteğiyle gelişen Django, iki temel zorlukla baş edebilecek biçimde tasarlanmıştır: Kısa zamanda bitirilmesi gereken projeler ve o projeleri yazan deneyimli web geliştiricilerinin sıkı gereksinimleri. Bu nedenle, Django'yla yüksek performanslı, şık web uygulamalarını hızlıca yazmak mümkündür.

Web geliştirme sürecindeki tekrarları ortadan kaldırarak, kendini tekrar etmeme presibine uygun olarak geliştirme yapmanızı sağlar.
